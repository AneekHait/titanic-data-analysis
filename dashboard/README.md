# Dashboard

Two self-contained output generators:

| File | Output | What it is |
|---|---|---|
| [generate.py](generate.py) | [index.html](index.html) | Interactive Chart.js dashboard (dark + light themes) |
| [generate_pdf.py](generate_pdf.py) | [titanic_eda_report.pdf](titanic_eda_report.pdf) | Classical chart-heavy EDA report (fpdf2) |

For the polished narrative report aimed at stakeholders, see [reports/](../reports/) instead.

---

## The HTML dashboard

[**Open dashboard/index.html**](index.html) in any modern browser. Single file, ~60 KB, no build step.

### Sections

| # | Section | What it shows |
|---|---|---|
| 1 | Headline Numbers | KPIs, marginal survival breakdowns, "how to read this dashboard" glossary |
| 2 | Feature Predictive Power | Effect-size ranking on a 0–1 scale (Cramer's V + \|r\|) |
| 3 | Odds Ratios | 8 headline contrasts with 95% CIs and Fisher p-values |
| 4 | Survival Rates | Overall + by sex / class / port with Wilson 95% CI error bars |
| 5 | Class × Sex | Joint heatmap (RdYlGn) — the most extreme survival gap in the dataset |
| 6 | Demographics | By age, fare, title, family size |
| 7 | Distributions | Underlying age + fare histograms |
| 8 | Lifeboat Reality Check | The mechanism that everything else predicts |
| 9 | Statistical Tests | Chi-square, t-tests, ANOVA with p-values |
| 10 | Numerical Correlations | Pearson r heatmap (hover for exact values) |
| 11 | Missing Values | Column-by-column with severity tags |
| 12 | Key Insights | Plain-English summary of what the numbers say |

### Interactivity

- **Hover** any bar, cell, or chart point to see exact values, sample sizes, and 95% CIs.
- **Sidebar nav** highlights the current section on scroll (scroll-spy).
- **Theme toggle** (top right) switches between dark and light. All chart colours, label colours, and heatmap palettes update in place.
- **Mobile responsive** — sidebar collapses to a hamburger menu under 768px.

## The classical PDF

[titanic_eda_report.pdf](titanic_eda_report.pdf) — 25 pages, generated by `generate_pdf.py` via fpdf2. Each section pairs a chart with a table, and most pages include a "What this means in plain English" callout.

It's the **older format** — kept because some readers prefer print and because it serves as a reproducible alternative to the editable DOCX.

PDF chart images live in [_pdf_charts/](_pdf_charts/) (gitignored). They're regenerated whenever `generate_pdf.py` runs.

## Rebuilding

```bash
# Dashboard only
make dashboard
# or:
python dashboard/generate.py

# Classical PDF only
make pdf
# or:
python dashboard/generate_pdf.py

# Both
make all
```
